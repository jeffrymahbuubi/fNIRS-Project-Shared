# fNIRS Dataset

This folder contains the fNIRS (Functional Near-Infrared Spectroscopy) dataset. There are two classes in the dataset: Healthy and Anxiety subjects. Each subject folder contains the metadata for each recording and the fNIRS signals in both raw and processed formats.

Download the `raw.zip` file containing *Raw Data Files and Imported fNIRS Processed with HOMER 3 MATLAB* from this [Google Drive link](https://drive.google.com/file/d/1LqChZQys9r4KfQT4_Cf8VmYA9WJaHqDw/view?usp=drive_link). You will be prompted to request permission. Please send your request to this email: `11208120@gs.ncku.edu.tw`

## Dataset Structure

### Raw Data Files and Imported fNIRS Processed with HOMER 3 MATLAB

- **Raw Data Files and Imported fNIRS Processed with HOMER 3 MATLAB**
  - **Raw Data Files**: The `.nirs` and `.snirf` files contain the raw fNIRS measurement data.
  - **Processed Data Files**: The `.csv` files contain concentration data (HbO - Oxyhemoglobin, HbR - Deoxyhemoglobin, and HbT - Total Hemoglobin) that have been preprocessed from the raw data using MATLAB HOMER3 tools.

## Task Information

- Each subject may contain subfolders corresponding to different cognitive tasks completed by the subject:
  - **GNG**: Go/No-Go task
  - **1backWM**: 1-back Working Memory task
  - **VF**: Verbal Fluency task
  - **SS**: Serial Subtraction task

- You will notice that some subjects only contain two task folders. For algorithm development, only subjects who completed all four different tasks are used, and subjects who did not complete all four tasks are excluded.
